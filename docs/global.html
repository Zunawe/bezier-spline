weightsCallback(i, knots) → {number}

A function that calculates the weight to be assigned to the "speed" of curve `i`.
Parameters:
Name Type Description
i number The index of the knot at the end of the curve to be weighted. Integers on [1, n-1].
knots Array.<Array.<number>> The knots provided to the spline. More specifically, each point is a `vecn` (see module of the same name).
Source:
  • index.js, line 6
Returns:
The scalar that will modify the "starting speed" at the start of this curve. Higher numbers means the curve is more strongly pulled in the direction of tangency at the first knot in the curve.
