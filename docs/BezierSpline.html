Gets all the points on the spline that match the query.
Parameters:
Name Type Description
axis number The index of the axis along which to solve (i.e. if your vectors are [x, y, z], 0 means solve for when x = value).
value number The value to solve for (i.e. a Bezier cubic is on the left of an equation and this value is on the right).

The easiest way to change the spline's knots. Knots are kept as special vector types, so setting an entire knot may break the program. Alternately, you can read the documentation for vecn and manipulate the knots yourself.
